# Cocktail Buddy

## A CLI Project

### What is Cocktail Buddy?
Cocktail Buddy is a CLI App that will tell you how to make your favorite classic cocktail!

### Installation Guide
- Clone repository
- Run "ruby bin/run"
- Follow the instructions and enjoy!
